# Fashion-MNIST-Image-Classification-using-Deep-Learning-CNN-
Fashion-MNIST Image Classification is a deep learning project using Convolutional Neural Networks (CNNs) to classify grayscale images of clothing items such as T-shirts, trousers, dresses, and shoes. It demonstrates an end-to-end workflow including data preprocessing, model training, evaluation, and visualization.
